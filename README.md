# Python Cybersecurity Quiz 

A Python command-line quiz game with 10 multiple-choice questions covering general cybersecurity concepts.

## Features
- 10 cybersecurity questions
- Multiple-choice answers (A/B/C/D)
- Input validation
- Score and percentage calculation
